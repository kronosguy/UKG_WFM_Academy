# Academy Courses & Learning Paths

## Welcome to the Academy

The KronosGuy Academy provides comprehensive, production-grade training for UKG Workforce Management systems. Our structured courses blend theory with hands-on labs to prepare you for real-world success.

## Course Structure

### Learning Levels

All academy content is organized into three progressive levels:

#### Level 1: Foundation
*Perfect for: New users, career changers*

- Foundational concepts and terminology
- System navigation and basics
- Common daily tasks
- Prerequisite for all other learning
- 4-12 hours per path

#### Level 2: Intermediate
*Perfect for: Experienced users, those expanding skills*

- Advanced features and workflows
- Complex scenarios and multi-step processes
- Optimization techniques
- Integration patterns
- 6-16 hours per path
- Requires Level 1 completion

#### Level 3: Advanced
*Perfect for: Power users, administrators, specialists*

- Complex customization and configuration
- Architecture and design patterns
- Performance tuning and optimization
- Enterprise-scale operations
- 8-20 hours per path
- Requires Level 2 completion

### Course Format

Each course includes:

```
Course Structure
├─ Modules (typically 3-8 per course)
│  ├─ Lessons (3-5 per module)
│  │  ├─ Instructional Content (video or text)
│  │  ├─ Checkpoints (quick quizzes)
│  │  └─ Resources (guides, templates, code)
│  ├─ Module Checkpoint (comprehensive quiz)
│  └─ Hands-On Lab (apply skills)
├─ Practice Quizzes
├─ Final Assessment
├─ Downloadable Resources
└─ Certificate (upon completion)
```

## Hands-On Labs

### Lab Types

#### Guided Labs
Structure: Step-by-step instructions with immediate feedback

Example: "Creating Your First Schedule"
- Duration: 20 minutes
- Difficulty: Beginner
- Prerequisites: Scheduling Fundamentals module
- Objective: Create a simple 1-week schedule

#### Challenge Labs
Structure: Scenario description with goals, minimal instructions

Example: "Optimize Labor Budget While Meeting Coverage"
- Duration: 45 minutes
- Difficulty: Intermediate
- Prerequisites: Labor Analytics module
- Objective: Create schedule meeting coverage needs within budget

#### Scenario Labs
Structure: Real-world business situations

Example: "Handle Peak Season Scheduling Crisis"
- Duration: 90 minutes
- Difficulty: Advanced
- Prerequisites: All scheduling intermediate modules
- Objective: Manage 3-week peak period with staff constraints

## Certification Programs

### Certification Tiers

#### Associate Certification
**Earned by**: Completing any Foundation path

- Time & Attendance Operations Associate
- Workforce Scheduling Associate
- System Administration Associate

Requirements:
- Complete 4 foundation modules
- Pass final assessment (80%+)
- Valid for 2 years

#### Professional Certification
**Earned by**: Completing Foundation + Intermediate

- UKG WFM Scheduler Professional
- UKG WFM Administrator Professional
- UKG WFM Analyst Professional

Requirements:
- Complete all foundation modules
- Complete 4-5 intermediate modules
- Pass professional assessment (85%+)
- Valid for 3 years

#### Expert Certification
**Earned by**: Completing all levels

- UKG WFM Specialist Expert
- Requires completion of Foundation + Intermediate + Advanced
- Pass expert assessment (90%+)
- Valid for 3 years
- Earns Expert badge on profile
