# UKG Analytics Labs (DataCamp-Inspired)

## The "Learning-by-Doing" Methodology
These labs provide a structured environment to master the data layer of UKG Pro WFM. Every lesson includes a live terminal or SQL workbook context.

## Active Labs

### 1. SQL for UKG Reporting
- **Lab environment:** Synthetic UKG SQL instance.
- **Challenge:** Write complex joins to reconcile punch data vs pay rules.
- **Auto-Grader:** Validates the result set against the expected output.

### 2. Workforce AI Workbench
- **Lab environment:** Python Notebook.
- **Challenge:** Build a basic predictive model for staffing shortages using historical UKG data.

### 3. Data Governance Simulators
- Identify and fix data quality issues (missing badge IDs, orphans) in a mock production environment.
